# Mushroom Classification Project

## Overview
This project classifies mushrooms as edible or poisonous using Machine Learning techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Google Colab

## Dataset
The dataset contains different mushroom features used for classification.

## Features
- Data preprocessing
- Model training
- Prediction system
- Accuracy evaluation
- Visualization

## Machine Learning Algorithm
- Decision Tree
- Random Forest
- Logistic Regression

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Feature Encoding
4. Model Training
5. Prediction
6. Accuracy Testing

## How to Run
1. Open the notebook in Google Colab
2. Install required libraries
3. Run all cells

## Output
The model predicts whether a mushroom is:
- Edible
- Poisonous
